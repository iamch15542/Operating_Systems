## hw4

### Compile:  

**Series** : ```$ g++ -Wall -o hw4_Series hw4_Series.c -lpthread```

**Pi** : ```$ g++ -Wall -o hw4_Pi hw4_Pi.c -lpthread```

### Execute:

**Series** : ```$ ./hw4_Series```

**Pi** : ```$ ./hw4_Pi```

### Test:

**Series** : ```$ make clean && make && make test``` or ```./hw4_Pi```

**Pi** : ```$ make clean && make && make test2``` or ```./hw4_Pi```
